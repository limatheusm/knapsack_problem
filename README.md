## Instruções

### Utilizando javac
```sh
    $ javac Knapsack.java
    $ java Knapsack.java < mochila01.txt
```
## Arquivo de entrada
Seguem o padrão do arquivo "mochila01.txt", ou seja, 
<br>
n M <br>
pi <br>
vi <br>
<br>
onde:
<br>
n: |O|<br>
M : capacidade da mochila<br>
pi : vetor de pesos dos objetos<br>
vi : vetor de valores dos objetos<br>
<br>
##### Arquivo exemplo
4 30<br>
13 23 17 19<br>
23 29 27 25<br>
